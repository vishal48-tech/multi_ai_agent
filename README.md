# Multi AI Agent System 🤖
A sophisticated multi-agent framework featuring specialized AI agents with memory, human-in-the-loop capabilities, and advanced reasoning patterns. Built for complex task automation, interactive decision-making, and intelligent problem-solving.

# ✨ Features
🧠 **Persistent Memory** – Each agent maintains short-term and long-term memory for context-aware interactions

👥 **Human-in-the-Loop** – Seamlessly integrates human approval, feedback, and intervention points

🔍 **RAG Agent** – Retrieval-Augmented Generation for knowledge-intensive tasks

💻 **CodeAct Agent** – Executes code, analyzes results, and iterates on solutions

🔄 **ReAct Agent** – Reason + Act pattern for dynamic tool use and observation

🪞 **Reflexion Agent** – Learns from past mistakes through verbal self-reflection

📋 **Plan-and-Execute Agent** – Decomposes complex goals into actionable steps

🌳 **Tree-of-Thought Agent** – Explores multiple reasoning paths simultaneously
